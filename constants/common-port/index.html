端口号的范围是从1～65535。

其中1～1024是被RFC 3232规定好了的，被称作“众所周知的端口”(Well Known Ports)；

从1025～65535的端口被称为动态端口（Dynamic Ports），可用来建立与其它主机的会话，也可由用户自定义用途。

默认端口号 说明
21/tcp FTP（文件传输）协议代理服务器
22/tcp SSH（安全登录）、SCP（文件传输）
23/tcp Telnet（不安全的文本传送）
25/tcp SMTP Simple Mail Transfer Protocol (E-mail)
53 DNS域名解析服务
69/udp TFTP（Trivial File Transfer Protocol ）
80/tcp HTTP超文本传输服务
110/tcp POP3 Post Office Protocol (E-mail)
443/tcp 443/udp HTTPS（securely transferring web pages）加密的超文本传输服务
1080/udp 腾讯QQ，SOCKS代理协议服务器常用端口号
1433/tcp 1433/udp MS SQL SERVER数据库server
1434/tcp 1434/udp MS SQL SERVER数据库monitor
1521 Oracle数据库服务
2100 Oracle XDB FTP服务
3306 MYSQL
3389 Microsoft RDP微软远程桌面
5000 MS SQL Server使用的端口
6379 Redis
7001 WebLogic
8080 JBOSS，TOMCAT，Oracle XDB（ XML 数据库），HTTP协议代理服务器
9000 php的fastcgi端口
